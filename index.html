Welcome to My Responsive Digital Space

A Quick Introduction to My Design

Hello! I am Aleisha Peterson, and this site demonstrates my approach to modern, responsive web design. The entire page frame (header, navigation, and footer) is managed by CSS Grid, ensuring structural integrity across all devices. The primary content and the sidebar you see here are managed by CSS Flexbox, allowing them to sit side-by-side on large screens and stack vertically on mobile. This combination prevents excessive horizontal scrolling.

My focus is on structured, accessible content through clean HTML, which is then styled for consistency using the external `styles.css` file. Every headline, paragraph, and link on every page will look consistent, creating a unified user experience.

Fluid Layout Demonstration

Notice how the layout changes when you resize your browser window! The content smoothly reflows: the navigation links wrap, the main column and sidebar stack, and the overall container adjusts its width fluidly using percentage units. This is the power of a liquid, responsive layout. If you'd like to see an example of my interests, including a featured video, please visit the Interests Page.

Site Features

  • Responsive Frame: CSS Grid
  • Content Flow: CSS Flexbox
  • Consistency: Single CSS file
  • Special Layouts: Grid on Gallery, Table on Contact
